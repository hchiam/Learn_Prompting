# PromptSandbox.io

[PromptSandbox.io](https://PromptSandbox.io) est une plateforme de programmation visuelle basée sur les nœuds pour concevoir et exécuter des flux de travail avec les API OpenAI.

Explorez le dépôt sur [GitHub](https://github.com/eg9y/promptsandbox.io/).

Regardez la visite guidée dans cette [vidéo de démonstration](https://www.youtube.com/watch?v=CBPw7FXtaEU).

## Fonctionnalités

- Nœuds Polyvalents : Utilisez divers types de nœuds, y compris les API OpenAI, les boucles, les conditions et les E/S de texte, pour créer des flux de travail complexes.
- Organisation de Projets : Créez, gérez et accédez facilement à plusieurs projets au sein de la plateforme.
- Chat Dynamique : Interagissez avec et testez les flux de travail via une interface de chat intuitive.
- Galerie Collaborative : Explorez et partagez des projets avec la communauté dans la galerie de projets.
- Accès Ouvert : Profitez d'un accès gratuit, sans inscription, à la programmation visuelle avec les API OpenAI.
