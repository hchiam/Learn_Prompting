# PromptChainer

[PromptChainer](https://arxiv.org/pdf/2203.06566.pdf)(@wu2022promptchainer) est un outil de recherche pour l'ingénierie de prompts. Il se concentre sur l'enchaînement des prompts, et c'est l'
EDI de recherche le plus raffiné que j'ai trouvé.

## Fonctionnalités

PromptChainer dispose d'une vue en chaîne, ainsi que d'une vue de nœud individuel. Ils
fournissent également des fonctionnalités de débogage.
