# PromptSource 

[PromptSource](https://github.com/bigscience-workshop/promptsource)(@bach2022promptsource) est un outil de recherche pour l'ingénierie de prompts. Il a été utilisé dans le développement de
BLOOM(@scao2022bloom), un LLM multilingue bien connu.

## Fonctionnalités

PromptSource permet des tests automatisés sur des jeux de données, et fournit une interface
propre via streamlit.
