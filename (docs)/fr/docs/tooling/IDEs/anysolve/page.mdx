# AnySolve

[AnySolve](https://www.anysolve.ai) est un site web pour générer et partager des tâches principalement basées sur des prompts textuels.

## Fonctionnalités

Les tâches peuvent être créées dans un éditeur d'interface utilisateur. Les tâches créées peuvent être partagées avec d'autres utilisateurs afin que tout le monde puisse y accéder. Une API REST permet d'intégrer la tâche dans d'autres logiciels.
