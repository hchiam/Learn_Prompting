# OpenPrompt

[OpenPrompt](https://thunlp.github.io/OpenPrompt/index.html) est une bibliothèque Python
qui fournit diverses fonctionnalités pour construire et tester des prompts.

## Fonctionnalités

OpenPrompt prend en charge les modèles de huggingface, mais pas encore OpenAI. Il se concentre davantage
sur les jeux de données et les tests que sur l'enchaînement. OpenPrompt semble également prendre en charge
le soft-prompting via https://github.com/thunlp/OpenDelta.
