# Lang Chain

[LangChain](https://github.com/hwchase17/langchain/) est une bibliothèque Python
qui fournit diverses fonctionnalités pour construire et enchaîner des prompts.

## Fonctionnalités

LangChain permet de changer facilement de fournisseur, standardise les interactions avec les LLM,
et fournit des _agents_ pour des choses comme le raisonnement symbolique et les architectures MRKL(@karpas2022mrkl).

LangChain prévoit d'ajouter une interface utilisateur graphique à l'avenir.
