# The Forge AI

[The Forge AI](https://theforgeai.com/) est un puissant EDI de prompts pour construire et monétiser des applications. Vous pouvez utiliser différents types d'entrées configurables, définir des flux conditionnels, générer du texte et des images en utilisant différents modèles. (Beaucoup plus de fonctionnalités à venir).

## Fonctionnalités

Une fois l'application terminée, vous pouvez ensuite la partager en privé ou publiquement pour que n'importe qui puisse l'exécuter.
Les applications peuvent également être monétisées sur la place de marché en vendant des jetons ou via un abonnement.
