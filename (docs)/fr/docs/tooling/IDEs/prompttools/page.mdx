# PromptTools

[PromptTools](https://github.com/hegelai/prompttools/) est une bibliothèque open-source pour
expérimenter, tester et évaluer les prompts, les LLM et les bases de données vectorielles.

## Fonctionnalités

PromptTools offre un playground pour comparer les prompts et les modèles dans trois modes :

- Instruction
- Modèle de prompt
- Comparaison de modèles

En plus du playground, PromptTools propose un SDK pour écrire des tests
et des fonctions d'évaluation pour expérimenter et évaluer les prompts à grande échelle.

Vous pouvez consulter une vidéo tutorielle [ici](https://www.youtube.com/watch?v=cLGRqNI-nJU).
