# OpenAI DALLE IDE

[OpenAI DALLE IDE](https://labs.openai.com) est un site web pour générer des images avec DALLE-2, un modèle de texte vers image.

## Fonctionnalités

OpenAI DALLE IDE offre une expérience utilisateur fluide et est facile à utiliser. Il dispose d'une vue historique, vous permettant de voir les générations passées, et d'une fonctionnalité de collections pour organiser et partager vos générations. Il permet également l'édition d'images et l'outpainting.
