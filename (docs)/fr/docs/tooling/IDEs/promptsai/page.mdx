# Prompts.ai

[Prompts.ai](https://prompts.ai/) fournit un playground similaire à celui de GPT-3, mais avec quelques
fonctionnalités supplémentaires. Il n'est plus en développement.

## Fonctionnalités

En particulier, Prompts.ai fournit des outils pour les complétions simultanées de
différents prompts, plusieurs complétions à partir du même prompt, et une
fonctionnalité de conversation.
