# Human Loop

[Human loop](https://humanloop.com/) semble offrir un playground similaire à celui de GPT-3, mais avec quelques fonctionnalités supplémentaires. Ils travaillent actuellement avec davantage d'utilisateurs industriels.

## Fonctionnalités

1. Fournit un SDK simple pour enregistrer les requêtes GPT-3 et les retours des utilisateurs.
2. Fournit des retours sur les entrées utilisateur et découvre les problèmes que l'utilisateur pourrait manquer.
3. Les utilisateurs pourront enregistrer des signaux explicites et implicites via le SDK.
4. Les utilisateurs pourront facilement effectuer des tests A/B sur les modèles et les prompts grâce au moteur d'amélioration conçu pour GPT-3.
5. Les utilisateurs pourront comparer les prompts sur différents modèles, trouver le meilleur modèle et réduire les coûts.
