# Soaked 

[Soaked](https://soaked-prompts.vercel.app) est un EDI d'ingénierie de prompts qui
se concentre sur l'enchaînement visuel des prompts avec une interface graphique. Il est actuellement en
version bêta, mais le développement actif a cessé.

## Fonctionnalités

Soaked fournit une interface graphique pour enchaîner les prompts ensemble,
où vous pouvez facilement voir comment différents composants s'articulent. Il
fournit également une vue de prompt de plus bas niveau avec l'option de générer plusieurs
complétions à la fois. Il fournit aussi une estimation de coût pour les complétions. Il prend actuellement en charge GPT3 et nécessite une clé OpenAI pour être utilisé.
