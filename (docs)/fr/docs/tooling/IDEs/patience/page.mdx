# Patience

[Patience](https://www.patience.ai) est un site web pour générer des images avec différents
modèles de texte vers image.

## Fonctionnalités

Patience propose la création de dossiers, la sauvegarde de collections/historique, et plusieurs modèles différents au choix.
Il permet également la modification de divers paramètres de diffusion comme la taille et la méthode d'échantillonnage.
