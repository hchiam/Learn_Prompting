# Dust 

[Dust](https://dust.tt/) est un outil d'ingénierie de prompts conçu pour enchaîner les prompts. Ils fournissent une interface web pour écrire des prompts et les enchaîner.

Pour le moment, il a une courbe d'apprentissage abrupte par rapport à d'autres EDI d'ingénierie de prompts.

## Fonctionnalités

Dust fournit des outils robustes sous forme de nombreux "blocs" composables, pour des fonctions comme les requêtes LLM, les extraits de code et les recherches sur internet. Dust prend également en charge l'utilisation de jeux de données et le test automatique des prompts sur ces jeux de données.

La fonctionnalité actuelle de Dust se concentre sur l'enchaînement des prompts plutôt que sur l'itération d'un seul prompt.

Dust prend en charge plusieurs fournisseurs de modèles : (OpenAI, Cohere), et prévoit de prendre en charge HuggingFace et Replicate. Des clés API sont requises pour tous les fournisseurs.

Vous pouvez déployer des applications LLM construites dans Dust.
