# GPT-3 Playground

[GPT-3 Playground](https://beta.openai.com/docs/quickstart) est l'interface web pour l'API GPT-3 d'OpenAI. C'est probablement l'
outil d'ingénierie de prompts original que la plupart des gens ont utilisé. En tant que tel, il est
extrêmement bien connu.

## Fonctionnalités

GPT-3 Playground fournit une interface épurée avec quelques cadrans différents pour
modifier le comportement de GPT-3. Il donne également accès à différents modèles GPT
de tailles de paramètres et de prix variés. Ils fournissent des crédits gratuits, qui expirent
après 3 mois.
