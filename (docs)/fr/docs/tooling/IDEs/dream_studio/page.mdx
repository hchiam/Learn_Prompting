# Dream Studio

[Dream Studio](https://beta.dreamstudio.ai/dream) est un site web pour générer des images avec Stable Diffusion, un modèle de texte vers image.

## Fonctionnalités

La principale fonctionnalité de Dream Studio est la possibilité d'ajuster divers paramètres de configuration du modèle, tels que la hauteur/largeur de l'image générée et les étapes/algorithme d'échantillonnage. Il permet l'inpainting et l'outpainting.

Il dispose actuellement d'une vue historique sous-développée (il semble sauvegarder des versions pixelisées des images dans un cache de navigateur, mais il n'y a aucun moyen de les visualiser).
